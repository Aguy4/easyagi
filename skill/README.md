## 视频简介

演示在通用智能平台中创建与管理 **Skill**，供大模型节点与角色定义引用，扩展 Agent 可调用的工具与能力。

## Skill 配置要点（摘要）

1. 进入 Skill 界面 → 「新建」  
2. 输入自定义名称  
3. **安装命令**（仅支持 SkillHub）或**上传 Skill 压缩包**，二选一  
4. 填写功能描述并保存  
5. Agent 应用可将 Skill 所需 API Key 等写入环境变量  

完整模块介绍（含大模型、MCP、知识库、Token 统计）见 **[model-mcp-skill-knowladge](../model-mcp-skill-knowladge/README.md)**。
