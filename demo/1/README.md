## 视频简介

通过简单的工具开发最符合自己需求的智能体。

通过一个完整案例，演示从**新建应用**到**飞书下达任务**的单 Agent 开发全过程，任务示例为：整理美伊历史冲突及对股市的影响，生成 PDF 并发送给用户。

## 开发流程摘要

### 1. 创建应用

1. 新建应用，进入开发界面
2. 右上角「应用设置」→ 选择 **Agent 应用**

**Agent 应用 vs 非 Agent 应用：**

| 类型 | 数据行为 |
|------|----------|
| Agent 应用 | 实例全局共享一份应用数据，适合有状态、需记忆的智能体 |
| 非 Agent 应用 | 每次流程调用数据重新初始化，适合无状态业务 |

### 2. 环境变量

为 Agent 应用添加环境变量（如 Skill 所需的 API Key），避免在流程中硬编码敏感信息。

### 3. 节点选型

本案例使用：

- **飞书拉取消息** — 接收用户指令
- **大语言模型** — 推理、搜索、生成 PDF
- **飞书发送消息** — 返回结果给用户

### 4. 大模型配置

- 选择模型提供商
- 挂载 **网络搜索 MCP**
- 挂载 **操作本地电脑 MCP**（容器环境需配置代理地址）
- 开启**深度思考**模式

### 5. 数据流连接

```
飞书消息 → 大模型 → 飞书发送
```

若需查看大模型中间步骤，将 `step` 一并连至飞书发送节点。

### 6. 示例指令

> 你整理一下历史上美国和伊朗都发生过哪些冲突，对股市都产生了什么影响，尽量的详细，整理成 pdf，发给我

## 学习收获

- Agent 应用的创建与配置
- 飞书 + 大模型 + MCP 的典型串联方式
- 环境变量与深度思考的实际用法
