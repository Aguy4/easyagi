---

## 一、机器视觉

机器视觉分为 **模型** 与 **数据集** 两个子模块，分别为工作流中的 YOLO 节点提供推理模型与训练数据。

### 模型管理

1. 进入「机器视觉 → 模型列表」，点击「新建」
2. 填写必填项：自定义名称、模型描述；选填项便于后续维护
3. 保存后，在模型卡片点击「导入」，上传模型压缩包
4. 压缩包根目录须包含 `model.pt`，导入后即可在工作流 YOLO 节点中使用

### 数据集管理

1. 进入「机器视觉 → 数据集」，点击「新建」
2. 填写数据集名称、数据量、描述
3. 在卡片上「导入」数据集文件
4. 导入完成后可用于 YOLO 训练；支持使用物理机 GPU 加速训练

## 二、应用部署

将开发完成的智能体部署为可持续运行的服务。

### 部署流程

1. 在应用开发界面，左下角工具栏选择「**保存为版本**」，填写版本号与描述
2. 回到应用列表，打开应用卡片右下角「版本列表」
3. 找到目标版本，点击「**部署**」

### 部署详情（5 个标签页）

| 标签页 | 功能 |
|--------|------|
| **运行** | 执行工作流中定义的命令列表 |
| **运行结果** | 查看命令执行结果（含界面运行结果） |
| **界面** | 以拖拽设计的 UI 交互运行工作流 |
| **模型配置** | 列出智能体中的模型类节点，可修改配置后「重载」 |
| **接口介绍** | HTTP 调用工作流命令的 API 说明 |

### 多种调用方式

- **命令列表**：在运行页直接触发
- **可视化界面**：在界面页上传图片等，演示 YOLO 识别与标注结果
- **HTTP 接口**：结果经 API 返回调用方（不出现在运行结果页）
- **飞书消息**：若工作流含飞书消息接收节点，可通过飞书下达任务
