## 视频简介

演示一个简洁的**单 Agent** 实战：让智能体自动查找通用智能官网地址，完成页面截图，并将截图通过飞书发送给指定用户。

## 案例目标

- 理解单 Agent 应用的典型任务闭环：**接收指令 → 执行（含工具/MCP）→ 回传结果**
- 体验飞书作为任务入口与结果出口的使用方式

## 能力要点

| 能力 | 说明 |
|------|------|
| 信息检索 | Agent 定位并访问官网 |
| 截图 | 对目标页面进行截图 |
| 飞书集成 | 将截图作为消息发送到飞书 |

## 适合人群

已了解平台基础配置（大模型、MCP 等），希望快速看到一个可运行的单 Agent 端到端示例的开发者。
